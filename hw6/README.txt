---ARROWMANIA---

To set up:
1. Download ArrowMania.py and the four arrow .png files.
2. Open an iPython interpreter.
3. cd to the directory within which it is contained.
4. Type 'run ArrowMania'

To play:
5. Purpose of game is to maximize score.
6. Use arrow keypad to destroy boxes within target range.
7. Game ends when all lives are lost.

Have fun!
